## 用自己的话回答

- 什么是特征，什么是标签：
- 特征：是用来描述样本的“属性”或“输入变量”（如代码中的 Tenure、Complain）。

标签：是我们想要预测的目标“结果”或“输出变量”（如代码中的 Churn）。
- 为什么要保留测试集：为了评估模型的泛化能力。训练集用于让模型学习规律，而测试集用来模拟模型在“未见过的新数据”上的表现，防止模型只是“死记硬背”（过拟合），从而检验它是否真正学会了规律。
- 为什么83%准确率仍可能没有用：因为准确率在某些场景下具有欺骗性。例如在不均衡的数据集中（如90%是不流失，10%是流失），如果模型全部预测为“不流失”，准确率也有90%，但对真正的流失用户毫无识别能力。此时更应关注召回率、精确率或F1-score等指标。
